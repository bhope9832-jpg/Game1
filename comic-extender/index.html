Comic Page Extender

Upload comic panels or a full page — the AI draws the next page (classic 6-panel layout) in the exact same art style, colors and tone.

Setup — free AI access (one-time, ~1 minute)

This app generates images with Google's Nano Banana models via the Gemini API. Heads-up: Google currently gives API keys zero free image-generation quota — the key's Google Cloud project must have billing attached (new accounts include $300 of free trial credit; one comic page costs a few cents, so the credit is effectively thousands of pages).

  1. Open aistudio.google.com/apikey and click “Create API key”.
  2. Attach billing to the key's project (AI Studio prompts you with a “Set up billing” link; new accounts get the $300 free trial credit).
  3. Paste the key below. It is saved only in this browser (localStorage) and sent only to Google.
